## Опис додатка
Назва: ShortyURL
Функціональність: Додаток призначений для ведення списку URL-посилань користувача з можливістю їх додавання, перегляду, вилучення та редагування. Система генерує скорочений вигляд довгих URL-адрес для зручного використання в мережі.

## Перелік сторінок додатку
Згідно із завданням, реалізовано наступні статичні сторінки:
1. Реєстрація користувача: форма з полями для імені, email, статі та дати народження.
2. Вхід до сайту: форма авторизації за email та паролем.
3. Профіль користувача: відображення даних у табличному вигляді.
4. Про додаток: сторінка з емблемою та коротким описом сервісу.
5. Робоча сторінка: основний інтерфейс для роботи з посиланнями.

## Використані технології
* HTML5 / CSS3: для структури та стилізації (шрифт Inter).
* Twitter Bootstrap: для створення адаптивного інтерфейсу та сітки.
